- 스티커의 z-index 존재. 렌더링 시 겹쳐진 상태에서 다른 영역 이 클릭되면 최상단으로 이동.

- 상단바와 하단바를 모두 제거하고, 팔레트를 눌렀을 때의 기능이. 메모가 없는 영역에서 우측 마우스 클릭으로 팝업으로 나타나도록. 그리고 거기에 Create가 있도록 함. 거기서 Screenshot과 Maximize 탭은 제거.

- (버그) 스티커의 어떤 영역에서건 우클릭 하면 팝업이 나타남. 현재 기능 맞음. 그런데 현재 기능이 텍스트가 보이지 않음.

- 쓸수 있는 영역은 테두리를 제외한 전부 이어야 함. 제목 같은건 따로 없음. 쓸 수 있는 영역 클릭 시 즉시 수정. focus 안된 상태에서는 클릭되면 기존에 커서가 있던 자리로 가고, focuse 된 상태에서는, 클릭된 곳으로 커서가 이동.

- 기본적으로 배경색은 필요하지 않음.

- 모서리 및 즉시 수정 기능. 아래의 기본 박스 UI에 대한 반영이 필요함.

상단 변 (두 줄로 표기) = 드래그 이동
좌측, 우측, 아래측 변 = 드래그로 크기 조절
테두리가 아닌 스티커 내 모든 영역 = 클릭 시 즉시 수정 가능
